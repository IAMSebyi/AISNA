# AI Agent Evaluation Workflow

This folder contains evaluation tests for the AISNA AI agents.

The current dataset is intentionally small and synthetic. Replace or extend
`fixtures/agent_evaluation_cases.json` when real labeled data becomes available.

Each evaluation case can include:

- `summary_output`: the summary generated by the summarizer agent
- `summary_expectations`: keyword, citation, and length checks
- `sentiment_output`: the sentiment output generated by the sentiment agent
- `sentiment_expectations`: expected overall and article-level sentiment labels
- `recommendation_expectations`: expected Buy / Hold / Sell recommendation

Run from the `backend` directory:

```bash
pytest tests/evaluation
```

These tests do not call OpenAI. They validate recorded outputs and the local
recommendation logic. For live model evaluation, generate agent outputs first,
store them in the fixture format, then run the same tests.
