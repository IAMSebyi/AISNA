# AISNA – AI Stock News Analyzer

## Project Description
AISNA (AI Stock News Analyzer) is an application that allows users to enter a stock ticker (e.g., AAPL, TSLA, NVDA) and receive:
- latest relevant news about the company
- an AI-generated summary
- sentiment analysis of the news
- a simple Buy / Hold / Sell recommendation

The project includes two AI agents:
- News Summarizer Agent – summarizes news articles about the selected company
- Sentiment & Recommendation Agent – analyzes sentiment and provides a simple recommendation

This project is developed as part of the Software Development Methods laboratory course at the Faculty of Mathematics and Computer Science, University of Bucharest.

## User Stories

US1 – As a user, I want to enter a stock ticker so that I can see information about a company.

US2 – As a user, I want to see the latest news about the selected company so that I can understand the current context.

US3 – As a user, I want to receive an AI-generated summary of the news so that I can save time.

US4 – As a user, I want to see whether the overall sentiment of the news is positive, negative, or neutral.

US5 – As a user, I want to receive a simple Buy / Hold / Sell recommendation so that I can interpret the information more easily.

US6 – As a user, I want to see the reasoning behind the recommendation.

US7 – As a user, I want to save favorite tickers so that I can track them more easily.

US8 – As a user, I want to see my search history so that I can quickly return to previously analyzed companies.

US9 – As a user, I want a clean and easy-to-use interface.

US10 – As a developer, I want to automatically test the main functionalities so that I can reduce errors.

US11 – As a developer, I want a CI/CD pipeline so that builds and tests run automatically.

US12 – As a developer, I want to evaluate the AI agents' output so that I can verify that the results are coherent and useful.
